What is NERVE

A Cloud FinOps and GreenOps orchestrator

Nick0 built NERVE from scratch during Hack'Europe at Paris Saclay, one of Europe's largest student hackathons. NERVE is a Cloud FinOps and GreenOps orchestrator. It monitors GPU usage across multiple cloud providers, spots waste in real time, and recommends optimizations to cut both cost and carbon emissions from AI workloads.

The frontend was a dashboard called Wattless, built in React and TypeScript. Wattless visualized spending, carbon metrics, and workload performance in a single view. The Node.js backend handled cloud provider integrations and the data aggregation layer powering it all.

What is Eve

An autonomous AI agent on top of NERVE

Eve is an autonomous AI agent built on top of NERVE. Describe the model you want in plain language, and Eve handles the rest: generates training code, picks the optimal GPU on cost and performance, deploys to the right cloud provider, and monitors the run end to end. If something breaks, she adjusts without human intervention.

The best analogy is AutoML, but fully conversational and fully autonomous. You could say "I need a sentiment classifier for French restaurant reviews, keep costs under fifty dollars" and Eve would figure out the architecture, the dataset pipeline, the training schedule, and the deployment target. Natural language in, deployed model out.

Why it matters

An AI system that creates other AI systems

Eve is an AI system that creates other AI systems. That means bridging ambiguous human intent and precise technical execution across the full stack: code generation, infrastructure selection, deployment, monitoring. All of it built solo at a hackathon.

The core insight: NERVE made AI infrastructure legible. Eve made it invisible. One gives you control, the other removes the need for it.

Status
