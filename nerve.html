NERVE + Eve

The platform that manages AI infrastructure.

01 — Overview

What is NERVE

NERVE is a full stack Cloud FinOps and GreenOps orchestrator that Nicolas Philippe designed and built from the ground up. FinOps, short for financial operations for cloud, is the discipline of optimizing GPU and compute spending across cloud providers. GreenOps extends that same philosophy to environmental impact, tracking and actively reducing the carbon footprint generated by AI workloads.

In practical terms, NERVE monitored GPU usage across multiple cloud providers, identified waste in real time, and recommended concrete optimizations to reduce both cost and emissions. The system ingested telemetry data from running workloads, processed it through a series of analytical pipelines, and surfaced actionable insights to the user.

The frontend was a dashboard called Wattless, built in React and TypeScript. Wattless visualized spending breakdowns, carbon metrics, and workload performance data in real time, giving engineers a single pane of glass into the true cost of their AI infrastructure. The backend ran on Node.js and handled everything from cloud provider integrations to the data aggregation layer that powered the dashboard.

02 — The agent

What is Eve

Eve is an autonomous AI agent that Nicolas built on top of the NERVE platform. The idea behind Eve was deceptively simple: describe the AI model you want in a natural conversation, and Eve handles the rest. No manual configuration, no infrastructure management, no guesswork about which GPU to rent.

Once a user described their requirements, Eve would generate the training code automatically, select the optimal GPU based on a cost and performance analysis, deploy the workload to the appropriate cloud provider, and monitor the entire training run from start to finish. If something went wrong, Eve could diagnose the issue and adjust parameters without human intervention.

The best analogy is AutoML, but fully conversational and fully autonomous. Where AutoML typically requires structured inputs and predefined search spaces, Eve worked from natural language alone. You could say "I need a sentiment classifier for French restaurant reviews, keep costs under fifty dollars" and Eve would figure out the architecture, the dataset pipeline, the training schedule, and the deployment target. She could handle the entire pipeline from natural language prompt to deployed, monitored model.

03 — Significance

Why it matters

NERVE proved that Nicolas Philippe could architect and build a complete production stack from scratch, end to end, with no team. That stack included a polished frontend dashboard, a robust backend API, integrations with multiple cloud provider APIs, and an autonomous AI agent sitting on top of everything. Very few engineers, especially self taught ones, can point to a single project that exercises all of those muscles simultaneously.

Eve specifically demonstrated something even more unusual: the ability to build AI systems that create other AI systems. Writing an agent that can autonomously generate training code, select infrastructure, deploy workloads, and monitor results is a non trivial engineering challenge. It requires deep understanding of both the AI/ML pipeline and the infrastructure layer that supports it. The fact that Eve worked through natural language made it even more technically demanding, because the agent had to bridge the gap between ambiguous human intent and precise technical execution.

The core insight: NERVE was about making AI infrastructure legible. Eve was about making it invisible. Together, they showed that Nick0 could build tools that abstract away enormous complexity without sacrificing control.

04 — Stack

Technical details

The system was split into two main codebases. The NERVE platform, which included Wattless and the backend, was a JavaScript/TypeScript project. Eve, the autonomous agent, was written in Python to take advantage of the ML ecosystem.

05 — Status

Current status

NERVE and Eve are not actively maintained. After completing the project, Nicolas Philippe shifted his focus to Friday, the autonomous Mac agent, and then to 20n Research Laboratory, where he now builds a universal brain for robots. The progression was natural: NERVE proved he could build complex cloud infrastructure, Eve proved he could build autonomous agents, and that combination of skills fed directly into the more ambitious work that followed.

The backend source code is publicly available on GitHub for anyone who wants to study the architecture or build on top of it.
